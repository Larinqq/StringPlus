# s21_string+

Реализация библиотеки обработки строк на языке C с дополнениями.

## О проекте

Проект представляет собой частичную реализацию основной библиотеки языка C для работы со строками — `string.h` — с добавлением собственных расширений. В рамках проекта разработаны:

- Частичная реализия библиотеки `string.h`
- Частичная реализация функции `sprintf`
- Дополнительные функции обработки строк (аналог методов класса String из C#)

## Технологии

- **Язык:** C (стандарт C11)
- **Компилятор:** gcc
- **Сборка:** Makefile
- **Тестирование:** библиотека Check
- **Покрытие кода:** gcov

## Реализованные функции

### Основные функции string.h

| Функция | Описание |
|---------|----------|
| `s21_memchr` | Поиск первого вхождения символа |
| `s21_memcmp` | Сравнение блоков памяти |
| `s21_memcpy` | Копирование блока памяти |
| `s21_memset` | Заполнение блока памяти |
| `s21_strncat` | Конкатенация строк с ограничением |
| `s21_strchr` | Поиск символа в строке |
| `s21_strncmp` | Сравнение строк с ограничением |
| `s21_strncpy` | Копирование строки с ограничением |
| `s21_strcspn` | Длина начального сегмента без указанных символов |
| `s21_strerror` | Сообщение об ошибке по номеру |
| `s21_strlen` | Длина строки |
| `s21_strpbrk` | Поиск первого совпадающего символа |
| `s21_strrchr` | Поиск последнего вхождения символа |
| `s21_strstr` | Поиск подстроки |
| `s21_strtok` | Разбиение строки на токены |

### Функции форматирования

| Функция | Поддерживаемые спецификаторы |
|---------|------------------------------|
| `s21_sprintf` | `c`, `d`, `f`, `s`, `u`, `%` |


### Специальные функции (расширения)

| Функция | Описание |
|---------|----------|
| `s21_to_upper` | Преобразование строки в верхний регистр |
| `s21_to_lower` | Преобразование строки в нижний регистр |
| `s21_insert` | Вставка подстроки в указанную позицию |
| `s21_trim` | Удаление начальных и конечных символов |
